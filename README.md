# Customer Review Sentiment Analysis

## Project Overview

This project is completed as part of the Horizon TechX Artificial Intelligence & Data Science Internship Program.

The project focuses on analyzing customer reviews and classifying them into Positive, Negative, and Neutral sentiments. It includes text cleaning, sentiment classification, visualization, dashboard creation, and report generation.

## Objective

The main objective of this project is to understand customer opinions from review text and ratings.

## Tools and Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
- VS Code
- GitHub

## Features

- Customer review dataset
- Text cleaning and preprocessing
- Sentiment classification
- Positive, Negative, and Neutral review analysis
- Rating distribution analysis
- Review length analysis
- Charts and visualizations
- Interactive Streamlit dashboard
- Final report generation

## Project Workflow

1. Create or load customer review dataset
2. Clean review text
3. Remove special characters
4. Classify sentiments using rating and keyword logic
5. Generate sentiment summary
6. Create charts and graphs
7. Generate analysis report
8. Build Streamlit dashboard

## Visualizations

- Sentiment Count
- Rating Distribution
- Rating vs Sentiment
- Review Length Distribution
- Average Rating by Sentiment

## Dashboard

The dashboard allows users to filter reviews by sentiment and rating. It also shows summary cards, charts, filtered reviews, and insights.

## Key Insights

- Positive reviews show customer satisfaction.
- Negative reviews highlight issues such as poor quality, late delivery, or bad service.
- Neutral reviews represent average customer experience.
- Ratings and sentiments are strongly connected.
- Sentiment analysis helps understand customer feedback in a simple way.

## Conclusion

This project demonstrates how sentiment analysis can be used to understand customer opinions. It uses Python libraries for data processing, visualization, and dashboard creation.
