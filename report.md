# Customer Review Sentiment Analysis Report

## 1. Introduction

Sentiment Analysis is a Natural Language Processing technique used to identify emotions or opinions from text data. In this project, customer reviews are analyzed and classified into Positive, Negative, and Neutral sentiments.

## 2. Objective

The objective of this project is to analyze customer reviews and understand customer feedback using sentiment classification and data visualization.

## 3. Tools and Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
- Visual Studio Code

## 4. Dataset Description

The dataset contains customer reviews and ratings. Each review includes a review ID, review text, and rating. Additional columns such as clean review, sentiment, and review length are created during processing.

## 5. Data Cleaning and Preprocessing

The review text is cleaned by converting it into lowercase and removing special characters. Missing values and duplicate records are also checked and handled.

## 6. Sentiment Classification

Reviews are classified into Positive, Negative, and Neutral sentiments using rating values and simple keyword-based logic.

Positive reviews include words such as good, great, excellent, amazing, happy, and satisfied.

Negative reviews include words such as bad, poor, worst, waste, disappointed, and damaged.

## 7. Visualizations Created

The following visualizations were created:

- Sentiment Count
- Rating Distribution
- Rating vs Sentiment
- Review Length Distribution
- Average Rating by Sentiment

## 8. Dashboard Features

An interactive Streamlit dashboard was created for better visualization. The dashboard includes:

- Sentiment filter
- Rating filter
- Total reviews card
- Positive reviews card
- Negative reviews card
- Neutral reviews card
- Average rating card
- Charts and graphs
- Filtered reviews table
- Download option

## 9. Key Insights

- Positive reviews indicate customer satisfaction.
- Negative reviews highlight problems related to product quality, delivery, or service.
- Neutral reviews show average customer experience.
- Rating distribution helps understand overall customer opinion.
- Sentiment analysis supports better decision-making.

## 10. Conclusion

This project successfully performs customer review sentiment analysis using Python. It includes data cleaning, sentiment classification, visualization, dashboard creation, and report generation.
